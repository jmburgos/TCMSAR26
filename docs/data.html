Data

This page describes the datasets available in the data/ directory. Datasets are organised into tabular (CSV), spatial vector (GeoPackage), and raster (GeoTIFF) files.

Downloading the data

Run the code chunk below once to download all course datasets into a local data/ directory. Files that already exist are skipped.

options(timeout = 300)  # large files may need extra time

base_url <- "https://raw.githubusercontent.com/jmburgos/TCMSAR26/main/data"

files <- c(
  # Tabular
  "datras_2018_haul.csv",
  "datras_2018_length.csv",
  "is_smb_biological.csv",
  "is_smb_stations.csv",
  "minke.csv",
  "small_vms.csv",
  "smb_summary.csv",
  "smb_utbrteg.csv",
  # Spatial vector
  "helcom.gpkg",
  "ice_lines.gpkg",
  "ice_points.gpkg",
  "ice_polygon.gpkg",
  "iceland_coastline.gpkg",
  "iceland_contours.gpkg",
  "ices_ecoregions.gpkg",
  "ices_rectangles.gpkg",
  "nephrops_fu.gpkg",
  "OSPAR_intensity_Otter_2015.gpkg",
  # Rasters
  "Faroes_minbtemp.tif",
  "Iceland_currentsp.tif",
  "Iceland_maxbtemp.tif",
  "Iceland_minbtemp.tif",
  "nephrops.tif"
)

dir.create("data", showWarnings = FALSE)

for (f in files) {
  dest <- file.path("data", f)
  if (!file.exists(dest)) {
    message("Downloading ", f, " ...")
    download.file(paste(base_url, f, sep = "/"), destfile = dest, mode = "wb", quiet = TRUE)
  } else {
    message("Skipping ", f, " (already exists)")
  }
}

message("Done.")

Read with: rast("data/Iceland_maxbtemp.tif")

Iceland_minbtemp.tif

Description: Minimum bottom temperature raster for Icelandic waters. Values in °C. Same grid as Iceland_maxbtemp.tif; together they represent the seasonal range of bottom temperatures.

Read with: rast("data/Iceland_minbtemp.tif")
